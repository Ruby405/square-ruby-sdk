## Catalog Query Text

### Structure

`CatalogQueryText`

### Fields

| Name | Type | Description |
|  --- | --- | --- |
| `keywords` | `Array<String>` | A list of one, two, or three search keywords. Keywords with fewer than three characters<br>are ignored. |

### Example (as JSON)

```json
{
  "keywords": [
    "keywords3",
    "keywords4"
  ]
}
```
